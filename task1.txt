Условие: В веб-приложении есть выпадающий список с выбором стран и кодом телефона. Необходимо проверить корректность работы списка.
Ожидаемый результат: Описание позитивных и негативных тест-кейсов.

Позитивные тест-кейсы:
1. Проверка реагирования элемента на взаимодействие
Кликнуть на выпадающий список
Ожидание:
открылся список

2.Проверка реагирования элемента на взаимодействие
Навести курсор на список
Ожидание:
открылся список

3. Проверка содержания списка
Просмотреть список
Ожидание:
В списке присутствуют только названия стран и соответствующие им коды

4. Проверка прокрутки
Просматриваем список
Ожидание:
Можем прокрутить список до конца

5. Проверка упорядоченности значений
Ожидание:
Список отсортирован по какому-либо признаку, упрощающему поиск

6. Проверки реагирования списка на прекращение взаимодействия с ним
Убрать курсор с открывшегося списка
Ожидание:
список свернулся

7. Проверки реагирования списка на прекращение взаимодействия с ним
Кликнуть на другое место экрана
Ожидание:
список свернулся

8. Проверка поиска в реальном времени
начинаем вводить название существующей страны
Ожидания:
Выводятся строки с названиями стран, соответствующие текущему вводу;

9. Проверка существования корректного кода для существующей страны
Вводим полностью название существующей страны
Ожидание:
список выводит единственную строчку с корректным кодом страны

10. Проверка корректности работы автозаполнения
Кликаем по выведенной строке
Ожидаение:
поле нажимается, реагирует на нажатие и форма автозаполняется кодом соответствующей страны

11. Проверить корректность отображения списка
Открыть список на разных браузерах и устройствах
Ожидание:
Список открывается, не искажается, не блокируется другими элементами, просматривается полностью

12. Проверка реакции поля на изменение ввода
Выбираем в списке код, соответствующей существующей стране, затем начинаем набирать название другой страны
Ожидание:
список выведет актуальные варианты заполнения, при клике на новую страну значение сменится


Отрицательный тест-кейсы:
1. Вводим несуществующую страну
Ожидание:
список выдает пустое значение, сообщение об ошибке ввода

2.Оставляем поле пустым
Ожидание:
Если поле обязательно  к заполнению, выводится предупреждение об обязательности заполнения

3. Вводим символы, отличные от букв
Ожидание:
список выдает пустое значение, сообщение об ошибке ввода

4. Вводим сразу несколько стран
Ожидание:
список выдает пустое значение, сообщение об ошибке ввода


Возможные баги:
- список не реагирует на нажатие/наведение;
- допускается ввод некорректного\пустого значения;
- несовпадение страны и кода;
- страны дублируются;
- код стран не соответствует принятому формату;
- строка не нажимается;
- при нажатии не происходит автозаполнение формы;
- список не отсортирован;
- не видно весь список - не умещается в окне на каком-либо типе устройств, в каком-либо браузере;
- в списке присутствуют несуществующие страны
